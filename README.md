This repo contains scripts to reproduce the numerical results in [1].

test_intro.m is for numerical experiments in Sec.1

test_multi.m is for Fig. 2.1

test_gaussian.m is for Fig. 2.2

test_conv.m if for Table 3.1 and 3.2

test_nlevp.m is for Fig. 3.1 and 3.2

test_stab.m is for Table 4.1 and 4.2

[1]: Nian Shao and Yuji Nakatsukasa. ``Convergence analysis of a nonlinear eigensolver based on rational approximation of the resolvent.''
